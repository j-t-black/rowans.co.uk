<objective>
## Goal
Configure the full Nuxt/NuxtUI/Tailwind framework stack so that the design system from Figma is handled at the root level — fonts, colors, spacing, typography scales, and component defaults all flow from framework configuration rather than per-page scoped CSS.

## Purpose
The site has ~95% scoped CSS with the same values duplicated across 20 pages. The Figma designs show a highly consistent, minimal design system (red titles, white body, black bg, monospace font, bordered dark cards). By configuring the framework to express this system natively, subsequent page work becomes "fix the 10%" rather than "rebuild everything."

## Output
- `app/assets/css/main.css` — expanded @theme tokens + CSS variables covering all Figma design tokens
- `app.config.ts` — NuxtUI component defaults matching Figma patterns (buttons, cards, inputs)
- Design token reference documented in plan SUMMARY for use in later phases
</objective>

## Design System Extracted from Figma Screenshots

### Colors
| Token | Hex | Usage |
|-------|-----|-------|
| bg-primary | #000000 | All page backgrounds |
| bg-card | #17181a / #1a1a1a | Cards, buttons, form inputs, contact boxes |
| text-primary | #ffffff | All body text |
| text-heading | #ff0000 (or close) | All page titles |
| text-link | #ff0000 | Inline links ("here"), CTAs |
| border-card | #ffffff / light gray ~40% | Card borders, input borders, button outlines |

### Typography (all JetBrains Mono)
| Token | Desktop Size (approx) | Usage |
|-------|----------------------|-------|
| title | 48-80px | Page headings ("Launch Parties", "Kids parties", etc.) |
| subtitle | 24-32px | Section subheadings ("25-100 people", "We can host your event") |
| body | 16-20px | Body copy, descriptions, list items |
| label | 14-18px | Form labels, small text, captions |
| button | 16-20px | Button text, CTA text |

### Layout Patterns
- Content: centered, text-align center
- Section spacing: ~40-80px vertical gaps
- Cards: dark bg, 1px light border, slight border-radius (~4-8px)
- Buttons: bordered boxes, centered text, ~padding 12-20px
- Forms: dark bg inputs with light borders
- Background images: dark atmospheric photos with implicit overlay on some pages

### Breakpoints (standardised)
- Mobile: 407px
- Tablet: 768px
- Desktop: 1280px
- Large desktop: 1920px

    - Dark-only color scheme

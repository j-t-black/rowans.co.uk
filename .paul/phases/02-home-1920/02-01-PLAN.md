<objective>
## Goal
Fix the home page to pixel-perfect match the Figma design at 1920px viewport width. Every section — Hero, Welcome, Party, Eats+Drinks, Audio, Visit Us, Map, Footer — must match the Figma Desktop frame (node 2:3, 1920x7328px) exactly.

## Purpose
The site was built and tuned at smaller viewport sizes by dragging the browser window. At 1920px (the Figma's native Desktop breakpoint), elements are undersized, incorrectly spaced, and don't match the stakeholder's design. This plan corrects the 1920px breakpoint first, then subsequent plans work down to mobile.

## Output
- `app/pages/index.vue` updated with correct 1920px styling
- All sections visually match `/tmp/comparison/figma-correct-1920.png` at 1920x1080 viewport
</objective>
